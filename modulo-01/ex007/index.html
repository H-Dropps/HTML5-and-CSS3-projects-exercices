Principais formatações:

Negrito

Usando <b> deixa em negrito na forma não semântica

Usando <strong> deixa em negrito na forma semântica

Italico:

Usando <i> deixa em italico na forma não femantica

Usando <em> deixa em italico na forma semântica

Marca texto:

Podemos também criar um texto marcado usando a tag MARK

texto grande: Texto grande

texto menor: texto pequeno

Texto deletado:

Podemos marcar um texto como deletado para indicar q deve ser lido, mas não considerado.

Texto inserido:

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. (semântica)

Existe também o sublinhado com a tag U. (não semântico)
